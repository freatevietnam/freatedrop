# Security policy

If you discover a security vulnerability in this project, please do not disclose it publicly right away.

## Reporting a vulnerability

- Open a private security report through your hosting platform if available.
- If private reporting is not available, contact the maintainer directly before publishing details.
- Include steps to reproduce, impact, and any suggested remediation if possible.

## Response expectations

The maintainer should acknowledge reports promptly, assess severity, and coordinate a fix before public disclosure.
